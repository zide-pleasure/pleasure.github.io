然后就封装了cookie的获取、设置、删除的三个方法

我个人对cookie并没有什么好感，一是大小限制，貌似4K左右，我生性自由之人，最受不了限制了。好在HTML5的localStorage本地存储，有5M大小，而且IE浏览器从IE8开始就已经支持了。IE6/7浏览器尚不支持，但这并不影响我们在实际项目中使用。此话怎讲，IE6~7浏览器依旧采用传统的cookie保持信息，其他浏览器就使用HTML5的本地存储功能。
